# Evolution-Random-String
Displays the evolution of text program and uses random string to create the test cases.

evolution.py is the evolution of text program itself.

random string - output.py creates a random string with the output in a text file.

random string.py only creates the string.

10,000,000 characters - 775.txt contains a document with 10,000,000 characters. It took 775 generations to finish.

10,000 - 914,2,3 is a folder which contains three text files which each have a 100,000 characters inside. The number of generations take is also in the file name. They took an average of 914 and two thirds tries to generate it. It is written as 914,2,3 instead.
